# 📊 Superstore Sales Analysis using Python

## 📌 Project Overview

This project analyzes the **Superstore Sales Dataset** using Python in Google Colab. The objective is to visualize sales trends over time, compare product categories, and summarize business performance using different chart types.

The project demonstrates data preprocessing, time series analysis, category-wise analysis, chart formatting, and exporting results.

---

## 🎯 Objectives

* Analyze sales trends over time.
* Perform monthly and quarterly sales aggregation.
* Compare sales across product categories.
* Visualize category-wise sales distribution.
* Save charts as PNG images.
* Export a summary report in CSV format.

---

## 📂 Dataset

**Dataset:** Sample Superstore Dataset

The dataset contains information about customer orders, including:

* Order Date
* Ship Date
* Customer Details
* Product Category
* Sub-Category
* Sales
* Quantity
* Discount
* Profit
* Region

## 📊 Analysis Performed

### 1. Data Loading

* Imported the dataset into Google Colab.
* Displayed the first few rows.
* Checked dataset information and missing values.

### 2. Data Preprocessing

* Converted the **Order Date** and **Ship Date** columns into datetime format.
* Verified data types.

### 3. Time Series Analysis

* Daily sales trend.
* Monthly sales aggregation.
* Quarterly sales aggregation.

### 4. Category Analysis

* Total sales by product category.
* Category comparison using bar charts.
* Category share using pie charts.

### 5. Summary Report

Generated a summary including:

* Total Sales
* Average Sales
* Maximum Sale
* Minimum Sale
* Total Orders
